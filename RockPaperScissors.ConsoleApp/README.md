## Introduction
A classic console implementation of Rock Paper Scissors where you face off against the CPU. Pick your move each round and watch the result resolve in real time.
 
---
## ✨ Features
- **Player vs CPU Gameplay:** Choose between Rock, Paper, or Scissors each round and challenge the CPU's randomized pick.
- **Classic Rules:** Rock crushes Scissors, Scissors cuts Paper, Paper covers Rock, and matching picks result in a tie.
