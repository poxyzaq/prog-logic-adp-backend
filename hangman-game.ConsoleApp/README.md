## Introduction

A console-based Hangman game featuring a rich vocabulary of exotic Brazilian fruits. Test your word-guessing skills as you try to uncover the secret fruit name before the hangman is fully drawn.

## ✨ Features

- **Fruit-Themed Vocabulary**: A diverse word bank featuring 29 unique fruits, including exotic Brazilian varieties like Acerola, Jabuticaba, Cupuaçu, and many more.
- **Visual Hangman Drawing**: The gallows and stick figure are progressively drawn with each wrong guess, providing clear visual feedback on your remaining attempts.
- **Real-time Feedback**: Each guess reveals any matching letters in the word, while incorrect guesses increment the error counter and advance the hangman drawing.
- **Win/Loss Conditions**: The game ends when you correctly guess all letters in the word or when the hangman drawing is completed after 5 errors.
- **Case-Insensitive Input**: Automatically converts all guesses to uppercase for seamless matching with the secret word.
