## Introduction

A console-based number guessing game where you try to figure out a secret random number within a limited number of attempts. Choose your difficulty, use the hints, and protect your score.

---

## ✨ Features

- **Difficulty Selection:** Choose between three difficulty levels — each changes both the number of attempts and the range of the secret number.

- **Instant Feedback:** After each wrong guess, the game tells you whether the secret number is higher or lower.

- **Repeated Guess Validation:** The game tracks your previous guesses and prevents you from wasting attempts on duplicates.

- **Scoring System:** Start with 1000 points. Each wrong guess deducts points based on how far you are from the secret number.
